API de Consulta de Resultados de Análisis

Estimado cliente,

Nuestra API de Consulta de Resultados de Análisis tiene como objetivo proporcionarle acceso seguro y eficiente a los resultados de sus análisis. Para garantizar la seguridad de sus datos, hemos implementado un sistema de identificación único llamado UUID (Identificador Único Universal). Este UUID le permite acceder de manera exclusiva a los resultados de sus análisis, brindándole una experiencia personalizada y protegiendo la confidencialidad de su información.

¿Cómo funciona?

Para consultar los resultados de sus análisis, simplemente introduzca el UUID correspondiente al análisis que desea revisar. Una vez proporcionado el UUID, tendrá acceso a los siguientes tipos de análisis:

  • Autocorrelaciones
  • Detección de Outliers
  • Descomposiciones de Series Temporales

Realice la consulta utilizando el UUID de su análisis y acceda a los resultados de manera rápida y sencilla.
